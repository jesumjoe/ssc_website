# Student Council Application

This project is a React application for the Student Council to manage concerns and track status.

## Getting Started

To run the project locally:

1.  Current Project is tested in Node.js version 22.11.0 (LTS)
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Run the development server:
    ```sh
    npm run dev
    ```

## Features

-   Submit Concerns (with Geotagged Photos)
-   Track Status
-   Admin Portal

## Tech Stack

-   Vite
-   React
-   TypeScript
-   Tailwind CSS
-   Shadcn UI
-   Supabase
